Customising your overlay

You can make the overlay look however you want. The process is: create a custom version of the overlay file, put it in the same folder as the app, and it will be used automatically.

Step 1 — Read the guide

Open OverlayConfigGuideline.md — it's in the same folder as the app. It explains what data is available, includes example overlays to start from, and walks you through how to use an AI chat tool to build and tweak your design without needing to know how to code.

Step 2 — Pick a starting point

The guide contains ready-to-use example overlays. Copy one, save it as overlay_custom.html, and you're ready to customise it. If you want to start from the built-in overlay instead, download overlay.html from the GitHub page and rename it overlay_custom.html.

Step 3 — Customise it with AI

Open an AI chat tool (Claude, ChatGPT, etc.), paste in the contents of OverlayConfigGuideline.md, then paste your overlay_custom.html and describe what you want to change — colours, layout, font size, what information to show. The AI will give you an updated file ready to use.

Step 4 — Put it next to the app

Save overlay_custom.html in the same folder as DueTiltedStats.exe. The overlay will switch to your version straight away — no restart needed. If you ever want to go back to the default, just delete the file.

The latest version of the guide and example overlays are always available on the GitHub page.
