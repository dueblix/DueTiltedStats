Customising your overlay

The overlay is a plain HTML file you can edit freely. Drop a custom version next to the .exe and it will be served automatically.

Step 1 — Get the guide

Open OverlayConfigGuideline.md — it's in the same folder as the .exe. It documents the data endpoint, includes dummy payloads for testing, and provides reference templates to start from.

Step 2 — Create your overlay file

Copy overlay.html from the templates/ folder of the source repository and rename it overlay_custom.html. Or start from one of the minimal templates in the guide.

Step 3 — Customise it

Edit the file directly, or paste the guide and your current overlay_custom.html into an AI chat and describe what you want. The CSS :root block at the top of the built-in template lists every value you can change without touching the JavaScript.

Step 4 — Drop it next to the .exe

Save overlay_custom.html in the same folder as the .exe. The overlay reloads from your file automatically — no restart needed. Delete it to revert to the built-in overlay.

The latest version of OverlayConfigGuideline.md is always available in the project's GitHub repository alongside the source code.
